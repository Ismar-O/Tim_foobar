FOOBAR Social media app created using nodejs, mongoDB.
## Installation
To run the app, clone this repository and install the required dependencies.
Using following npm packages: 
    -express
    -path
    -body-parser
    -cookie-parser
After everything is succesfully installed, in cmd you need to execute a command 'node app.js' in a foler where file 'app.js' is stored to start a backend server.

#Features
App suports login and register connected with mongoDB data base. 
When logged in you can create and view posts in feed.
Site supports cookie sessions. When logged in a session begins.
